## Design Context

### Users
- Single user (developer/owner), uso pessoal
- Dois contextos: planejamento em casa (desktop/mobile) e execucao no treino (mobile, uma mao, entre series)
- Possibilidade de terceiros criarem planos para o usuario
- Job to be done: montar treinos, registrar sets/cargas em tempo real, acompanhar evolucao

### Brand Personality
- **Energetico**: transmitir motivacao e intensidade sem ser agressivo
- **Minimalista**: informacao essencial, zero ruido visual
- **Funcional**: cada elemento existe por um motivo — priorizacao brutal de usabilidade no contexto de academia (tela pequena, mao suada, pressa entre series)

### Aesthetic Direction
- Visual tone: clean com acentos vibrantes — superficies neutras com cor nos pontos de acao
- Cores base: lime eletrico (#84cc16 / #a3e635) como primaria, amarelo (#facc15 / #fde047) como accent, charcoal neutro como base
- Dark mode como default (contexto academia com telas escuras), light mode disponivel
- Anti-reference: nao parecer generico/corporativo — deve ter personalidade propria
- Tipografia: sem serifa, pesos contrastantes (bold para numeros/metricas, light para labels)
- Espacamento generoso — touch targets grandes para uso no treino

### Design Principles
1. **Mobile-first, gym-first**: toda decisao de layout parte do cenario "celular na mao entre series". Botoes grandes, inputs acessiveis, informacao escaneavel
2. **Data over decoration**: numeros (peso, reps, series) sao protagonistas. Hierarquia visual favorece metricas
3. **Minimal friction**: minimo de taps para registrar um set. Fluxo de treino ativo e a prioridade #1
4. **Energy through contrast**: usar cor e peso tipografico para criar energia, nao ilustracoes ou animacoes excessivas
5. **Progressive disclosure**: mostrar o essencial, revelar detalhes sob demanda (expandir historico, notas, etc.)

### Tech Constraints
- React 19 + TypeScript + Vite
- Sem framework CSS atualmente — a definir
- FontAwesome (solid) para icones
- Recharts para graficos de evolucao
- Sem component library (UI construida do zero)
